# Introduction 
Readme for thunderstore is inside of directory - for anyone wanting to see actual code & use it for your convenience, here it is.  
I did join the REPO Modding Discord Server (Nubez) so if have any questions / suggestions you're more than welcome to contact me.   
Code is a mess, I started by combining 2 different decompiled mods and went from there - since its a small project it doesn't bother me *too* much but I don't think its worth the effort to refactor.  
Also while I have over 7 years of C# experience, I have none with game-dev / modding so if you think you see something dumb, it probably is, but technically it works :D 

Current Functionality
- Show Teammates 
- Show Dead Teammates
- Show Enemies
- Show Items
- Show Cosmetic boxes
- Auto Explore all rooms in map
- Change colors for everything
- Can Zoom In & out of map
- Can Change position of map 

best way to modify settings is in the Game's Settings menu ('Nubez Minimap')
